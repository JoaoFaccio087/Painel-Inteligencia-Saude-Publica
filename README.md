# :bar_chart: Painel de Inteligência em Saúde Pública

## :rocket: Objetivo do Projeto

O **Projeto Extensionista Integrador** tem como objetivo o desenvolvimento de um **Painel de Inteligência em Saúde Pública** para o estado de Mato Grosso. O painel integra dados sobre **casos de dengue** e **cobertura vacinal** (BCG e Tríplice Viral D1) por município, utilizando um Data Warehouse implementado em **Supabase (PostgreSQL)**. As visualizações são realizadas em **Power BI**, facilitando a análise temporal e territorial para apoiar a gestão pública de saúde.

## :star2: Tema do Projeto

**Implementação de Inteligência de Negócios (Business Intelligence) na Saúde Pública**.

Este projeto visa proporcionar uma ferramenta eficiente para a **análise de dados de saúde pública** e a **tomada de decisões** baseada em dados precisos e atualizados sobre dengue e vacinação.
