# Apresentações (PITCH)

Esta pasta contém os arquivos relacionados às apresentações do projeto, realizadas durante o **Projeto Extensionista Integrador**. As apresentações foram feitas em diferentes momentos do projeto, incluindo os **PITCHs** realizados para mostrar o progresso e os resultados alcançados.

## Arquivos Contidos

- **Slides do PITCH**: Apresentações em **PowerPoint** ou **PDF** detalhando os objetivos do projeto, soluções apresentadas e resultados.
- **Vídeos**: Gravações dos PITCHs apresentados, onde o grupo defende o projeto perante os avaliadores e participantes.

## Objetivo

O objetivo desta pasta é manter os arquivos de apresentação organizados, oferecendo um registro visual e documental dos momentos importantes de defesa do projeto e suas principais atualizações.
