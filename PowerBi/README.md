# Power BI

Esta pasta contém todos os arquivos relacionados ao **Painel de Inteligência em Saúde Pública** desenvolvido no **Power BI**. O painel tem como objetivo apresentar dados sobre **casos de dengue** e **cobertura vacinal** (BCG e Tríplice Viral D1) por município no estado de Mato Grosso.

## Arquivos Contidos

- **.pbix**: Arquivos do Power BI utilizados para criar os dashboards.
- **Visualizações**: Gráficos, KPIs, e mapas interativos usados para análise.

## Objetivo

O objetivo desta pasta é manter organizados os arquivos **Power BI** que mostram a visualização dos dados processados, proporcionando insights para a gestão pública e análise de saúde pública em Mato Grosso.
